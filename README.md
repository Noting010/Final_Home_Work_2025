#Проект моделирует корзину товаров в магазине.
Есть базовый абстрактный класс Food, конкретные продукты Meat и Apple, интерфейс Discountable для расчёта скидок и класс ShoppingCart, который подсчитывает:

    общую сумму товаров без скидок;

    общую сумму товаров с учётом скидок;

    сумму всех вегетарианских товаров без скидки.

Красные яблоки получают повышенную скидку, остальные продукты — без скидки.

    Структура проекта
model/

  Food — абстрактный класс с полями:

    amount — количество в килограммах;

    price — цена за килограмм;

    isVegetarian — флаг вегетарианского продукта.

  Meat — наследник Food, не вегетарианский продукт, без скидки.

  Apple — наследник Food с полем colour, реализует скидку для красных яблок.

  Discountable — интерфейс с методом getDiscount().

    constants/

      Discount — константы скидок (например, RED_APPLE = 60.0).

      Colour — константы цветов яблок (RED, GREEN и т.д.).

service/

  ShoppingCart — класс корзины с методами подсчёта стоимости.

Main — точка входа, создаёт объекты продуктов, корзину и выводит результаты в консоль.
